# Home assignment - Meteo App

```
App requires server with report data which provides object

{
   id: string,
   temperature: number,
   unit: string
   city: string,
   date: string
}

unit: 'K', 'C', 'F'

App allows to:
- edit the report
- add a new report
- sort ascending by temperature, date, city
- filter by city
```

## Instalation

```

Install dependencies
npm install

Start live reloading
npm run dev

Running on port 5173
```
